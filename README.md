# Birthday Gift Site

一个可直接部署到 GitHub Pages 的生日礼物网页。

## 1. 自定义内容

- 在 `index.html` 里修改标题文案和信件内容。
- 在 `script.js` 里修改生日日期：
  - `birthdayMonth`
  - `birthdayDay`

## 2. 本地预览

你可以直接双击 `index.html` 预览，或使用任意本地静态服务器。

## 3. 部署到 GitHub Pages

1. 新建一个 GitHub 仓库，例如：`birthday-gift`。
2. 把当前目录文件推送到仓库。
3. 打开仓库 `Settings` -> `Pages`。
4. 在 `Build and deployment` 下选择：
   - `Source`: Deploy from a branch
   - `Branch`: `main` / `root`
5. 保存后等待 1~3 分钟，访问页面地址：
   - `https://你的用户名.github.io/birthday-gift/`
